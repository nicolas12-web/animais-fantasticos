# 🐾 Animais Fantásticos

Projeto desenvolvido durante o curso de **JavaScript Completo ES6** da **Origamid**, com o objetivo de praticar manipulação do DOM, eventos, animações e modularização utilizando JavaScript moderno.

## 📖 Sobre o projeto

O **Animais Fantásticos** é uma aplicação web responsiva que apresenta informações sobre diferentes espécies de animais por meio de uma interface dinâmica e interativa.

Durante o desenvolvimento foram implementadas diversas funcionalidades utilizando JavaScript puro, seguindo boas práticas de organização e reutilização de código.

## ✨ Funcionalidades

- ✅ Navegação suave (Smooth Scroll)
- ✅ Sistema de Tabs
- ✅ Accordion (Perguntas Frequentes)
- ✅ Modal de Login
- ✅ Tooltip
- ✅ Dropdown Menu
- ✅ Menu Mobile
- ✅ Outside Click
- ✅ Animações ao Scroll
- ✅ Contador animado
- ✅ Funcionamento baseado em dias e horários
- ✅ Consumo de dados com Fetch API
- ✅ Código organizado em módulos ES6

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

## 💻 Como executar

Clone o repositório:

```bash
git clone https://github.com/nicolas12-web/animais-fantasticos.git
```

Entre na pasta do projeto:

```bash
cd animais-fantasticos
```

Abra o arquivo `index.html` no navegador ou utilize a extensão **Live Server** do VS Code.
